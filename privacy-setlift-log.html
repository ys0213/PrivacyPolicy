Privacy Policy

This Privacy Policy explains how SetLift Log handles user data.

1. App Overview

SetLift Log is a workout set logging app. It helps users record exercise sets, reps, optional weight, calculated volume, daily workout summaries, and calendar-based workout history.

The app does not require an account, login, or server connection for workout logging features.

2. Workout Data Stored on Device

SetLift Log allows users to enter and save workout-related records on their device. This may include:

  • Exercise names added by the user
  • Workout dates
  • Set records
  • Repetition counts
  • Optional weight values
  • Calculated workout volume
  • Daily workout summaries
  • Calendar workout history
  • App settings, such as weight unit and selected exercise

User-entered workout data is stored locally on the user's device. SetLift Log does not upload user-entered workout records to our servers.

3. Data Collected for Advertising

SetLift Log uses Google AdMob to display advertisements. Google AdMob may collect and process data for advertising, analytics, fraud prevention, diagnostics, and service improvement purposes.

Data processed through Google AdMob may include:

  • Advertising ID
  • Device information, such as device model and operating system version
  • IP address
  • App interaction data
  • Advertising-related data
  • Crash logs, diagnostics, and performance data

Ads may be personalized or non-personalized depending on the user's region, consent choices, device settings, and Google advertising settings.

4. Purpose of Data Use

Data may be used for the following purposes:

  • To save workout records locally on the user's device
  • To show workout set history by date
  • To calculate and display total sets, total reps, and workout volume
  • To allow users to edit or delete saved set records
  • To remember app settings, such as weight unit and selected exercise
  • To display advertisements through Google AdMob
  • To help improve app stability and advertising reliability
  • To help prevent fraud, abuse, and invalid advertising activity

5. Data Sharing

User-entered workout records stored locally on the device are not shared with us and are not stored on our servers.

Advertising-related data may be collected, processed, or shared by Google AdMob according to Google's own privacy practices and advertising policies.

6. Data Retention

User-entered workout records and app settings are stored locally on the user's device.

This locally stored data remains on the device until the user deletes it in the app, clears the app's storage/data through Android system settings, or uninstalls the app.

Because SetLift Log does not store user-entered workout records on our servers, we do not retain user-entered workout records outside the user's device.

7. Data Deletion and User Controls

Users can manage and delete saved workout data within SetLift Log.

SetLift Log provides the following user controls:

  • Users can edit an individual set record using the Edit button.
  • Users can delete an individual set record using the Del button.
  • Users can delete all set records for a selected date using the Clear Date button.
  • Users can delete today's set records using the Clear button in the Today section.
  • Users can add, edit, or delete exercise names using the Manage section.
  • Users can change the weight unit setting between lb and kg.

Users can also delete all locally stored app data by clearing the app's storage/data through Android system settings or by uninstalling the app.

Because SetLift Log does not require an account and does not store workout records on our servers, there is no server-side account data or server-side workout history to delete.

8. Third-Party Services

SetLift Log uses Google AdMob for advertising. Google may collect and process data according to Google's own privacy policy, advertising policies, and service terms.

9. Health and Fitness Disclaimer

SetLift Log is intended for personal workout logging and fitness record keeping only.

The app does not provide medical advice, medical diagnosis, treatment recommendations, injury prevention advice, rehabilitation guidance, professional fitness coaching, emergency health services, or personalized exercise prescriptions.

Users should consult a qualified professional before starting or changing an exercise program, especially if they have any medical condition, injury, or health concern.

10. Children's Privacy

SetLift Log is not intended for children under the age of 13. We do not knowingly collect personal information from children under 13.

11. Security

User-entered workout records are stored locally on the user's device. Users are responsible for protecting access to their own device.

SetLift Log does not operate a server that stores user-entered workout records.
